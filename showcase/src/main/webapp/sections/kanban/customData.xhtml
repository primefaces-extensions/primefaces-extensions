Any custom key-value data attached to a KanbanItem via putData(key, value) is rendered as a data-* attribute on the card DOM element. This allows you to attach arbitrary metadata such as priority, assignee, due date, or story points to each card.

The data attributes are rendered on each card as visual badges (priority, assignee, due date, labels, story points) via the extender callback, and are also available for CSS attribute selectors (e.g. [data-priority="high"]) and JavaScript via element.dataset.*. Inspect a card with your browser's developer tools to see the data-* attributes on the DOM element.
