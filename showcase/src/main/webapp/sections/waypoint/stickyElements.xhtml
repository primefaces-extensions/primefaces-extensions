An example with a sticky menubar and "back to top" link. Try to scroll down and see how the menubar gets stuck at top and the link appears at bottom right. Scroll up and see how the menubar gets unstuck and the link disappears. The Waypoint component self is used in combination with pe:javascript. In the callback function we check the current scrolling direction and set "sticky" style class on the current waypoint object. ${showcase:getFileContent('/sections/waypoint/examples/example-stickyElements.xhtml')}
